
# userDetails Method
This method is used to get the current user details from the Okto SDK. It will only return the user details if the user is authenticated.

## Parameters
None


## Example
```dart copy
import 'package:flutter/material.dart';

class UserDetailsScreen extends StatefulWidget {
  const UserDetailsScreen({super.key});

  @override
  State<UserDetailsScreen> createState() => _UserDetailsScreenState();
}

class _UserDetailsScreenState extends State<UserDetailsScreen> {
  UserDetails? _userDetails;
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: TextButton(
            onPressed: () async {
              final userDetails = await okto.userDetails();
              setState(() {
                _userDetails = userDetails;
              });
            },
            child: const Text('Get user details')),
      ),
    );
  }
}

```
