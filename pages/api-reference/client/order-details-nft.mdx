Get order details of all the NFTs transferred and received by the user.

<ColumnLayout>
    <>
        ## Headers
        `Authorization` : string <br />
        Authorization token received from Okto.

        ---

        ## Query Parameters
        `page` : number <br />
        Page number for pagination. Default is 1.

        ---
        `size` : number <br />
        Number of records per page. Default is 10.

        ---
        `order_id` : string <br />
        Order id of the NFT.

        ---
    </>
    <>
        ## Code
        ```bash copy
        curl -X GET '${baseUrl}/api/v1/nft/order_details?order_id=dd784ad9-68f8-49b9-87dc-13723e933eab&page=1&size=500'
        -H 'x-api-key: YOUR_CLIENT_API_KEY'
        -H 'accept: application/json' 
        -H 'Authorization: Bearer AUTH_TOKEN'
        ```
