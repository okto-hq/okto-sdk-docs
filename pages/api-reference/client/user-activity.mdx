Get user portfolio activity

<ColumnLayout>
    <>
        ## Headers
        `Authorization` : string <br />
        Authorization token received from Okto.

        ---
        
        ## Query Parameters
        `limit` : number <br />
        Number of records to return. Default is 10.

        ---
        `offset` : number <br />
        Number of records to skip. Default is 0.

        ---
    </>
    <>
        ## Code
        ```bash copy
        curl -X GET '${baseUrl}/api/v1/portfolio/activity?limit=10&offset=1'
        -H 'x-api-key: YOUR_CLIENT_API_KEY'
        -H 'accept: application/json' 
        -H 'Authorization: Bearer AUTH_TOKEN'
        ```
