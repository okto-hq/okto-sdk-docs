Get an array of all the whitelisted networks on your API key.
> How to whitelist a network? [Click here](/#supported-networks-and-tokens)

<ColumnLayout>
    <>
        ## Headers
        `Authorization` : string <br />
        Authorization token received from Okto.

        ---

        ## Parameters
        None
    </>
    <>
        ## Code
        ```bash copy
        curl -X GET '${baseUrl}/api/v1/supported/networks' 
        -H 'x-api-key: YOUR_CLIENT_API_KEY'
        -H 'accept: application/json' 
        -H 'Authorization: Bearer AUTH_TOKEN'
        ```

        ## Response
        ```json
        {
            "status": "success",
            "data": {
                "network": [
                    {
                        "network_name": "POLYGON_TESTNET",
                        "chain_id": "80001"
                    }
                ]
            }
        }
        ```
    </>
</ColumnLayout>
