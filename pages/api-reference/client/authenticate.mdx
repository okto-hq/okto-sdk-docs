Authenticates the user and on successfull authentication, it will return the token of the user.  

BaseUrl:
- Sandbox : `https://sandbox-api.okto.tech`
- Staging : `https://3p-bff.oktostage.com`
- Production : `https://apigw.okto.tech`


<ColumnLayout>
    <>
        ## Parameters
        `id_token` : string <br />
        The id token of the user received from the google oauth2 provider.

        ---
    </>
    <>
        ## Code
        ```bash copy
        curl -X 'POST' '${baseUrl}/api/v1/authenticate' -H 'accept: */*' 
            -H 'x-api-key: YOUR_CLIENT_API_KEY'
            -H 'Content-Type: application/json' -d '
                {
                    "id_token": "YOUR_TOKEN_ID"
                }
            '
        ```

        ## Response
        ```json
        {
            "status": "success",
            "data": {
                "auth_token": "ACCESS_TOKEN",
                "message": "success",
                "refresh_auth_token": "REFRESH_TOKEN",
                "device_token": "12d1a2676694d482dd453d1231ecdd8c0c87956ed646b3a8115e3ac7c42daf28"
            }
        }
        ```
    </>
