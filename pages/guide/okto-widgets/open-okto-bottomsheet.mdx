  Okto Bottomsheet is opened first when a new user `authenticates` with Okto and gets prompted to [`create wallet`](/guide/authenticate/create-wallet). 
  Now if you want the user to access this bottom sheet again, Okto provides [`showWidgetSheet`](/sdk-reference/react-native/showWidgetSheet) method to open the bottom sheet again.

    <Steps>
      ### Add Button to Open Bottom Sheet
      Let's add a button in `user-profile.tsx` to open the bottom sheet.

      ```tsx filename="user-profile.tsx" copy
      import { View, Button, Text } from 'react-native';
      import { useOkto, type OktoContextType } from 'okto-sdk-react-native';
      import React from 'react';

      const UserProfileScreen = () => {
        const { showWidgetSheet } = useOkto() as OktoContextType;

        return (
          <View>
            <Text>User Profile</Text>

            ...

            <Button
              title="Open Okto Profile"
              onPress={() => {
                showWidgetSheet();
              }}
            />
          </View>
        );
      };

      export default UserProfileScreen;
      ```

    <Steps>
      ### Add Button to Open Bottom Sheet
      Let's add a button in `user_profile.dart` to open the bottom sheet.

      ```dart filename="lib/screens/user_profile.dart" copy
      import 'package:flutter/material.dart';
      import 'package:okto_sdk_flutter/okto_sdk_flutter.dart';

      class OktoBottomSheet extends StatelessWidget {

        @override
        Widget build(BuildContext context) {
          return Scaffold(
            appBar: AppBar(
              title: Text('Okto Widget'),
            ),
            body: Center(
              child: Column(
                mainAxisAlignment: MainAxisAlignment.center,
                children: [
                  Text('Okto Bottom Sheet'),
                  ElevatedButton(
                    onPressed: () async {
                      await okto.openBottomSheet(context: context); // you can customize the bottom sheet by passing parameters.
                    },
                    child: Text('Open Okto Widget'),
                  ),
                ],
              ),
            ),
          );
        }
      }
      ```
