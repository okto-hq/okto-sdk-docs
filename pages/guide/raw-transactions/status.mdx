Now that the transaction is executed, we get a `jobId` after the API is successful. We can use the `jobId` to track the status of the transaction.

<Steps>
    ### Fetch Transaction Details
    We will get the transaction details using the [`raw-transaction-status`](/api-reference/client/raw-transaction-status) API. This API will return the status of the transaction and other transaction details.
    Create a new screen with the name `raw-transaction-status` and we will fetch the details here. 
    > Setup navigation to accept the `jobId` as a parameter. You can follow the same steps as we did for [Order Details](/guide/transfer-assets/order-details#fetch-order-details).

    <Tabs items={["React Native", "Flutter"]}>
        <Tabs.Tab>
            ```tsx filename="raw-transaction-status.tsx" copy
            import { useOkto, type OktoContextType} from 'okto-sdk-react-native';

            ```
        </Tabs.Tab>
        <Tabs.Tab>
            ```dart filename="raw_transaction_status.dart" copy
            import 'package:flutter/material.dart';
            import 'package:okto_sdk/okto_sdk.dart';

            class FetchRawTransactionScreen extends StatelessWidget {
              final String jobId;

              const FetchRawTransactionScreen({Key? key, required this.jobId}) : super(key: key);

              @override
              Widget build(BuildContext context) {
                return Scaffold(
                  appBar: AppBar(title: Text("Transaction Details")),
                  body: FutureBuilder(
                    future: OktoSdk.rawTransactionStatus(orderId: jobId),
                    builder: (context, snapshot) {
                      if (snapshot.connectionState == ConnectionState.waiting) {
                        return Center(child: CircularProgressIndicator());
                      } else if (snapshot.hasError) {
                        return Center(child: Text("Error: ${snapshot.error}"));
                      } else if (!snapshot.hasData) {
                        return Center(child: Text("No data"));
                      } else {
                        final transactionDetails = snapshot.data;
                        return Column(
                          crossAxisAlignment: CrossAxisAlignment.start,
                          children: [
                            Text("Network: ${transactionDetails.data.jobs[0].networkName}"), // change this 0 to some `index` inside some iterable class.
                            Text("Status: ${transactionDetails.data.jobs[0].status}"),
                            Text("Transaction Hash: ${transactionDetails.data.jobs[0].transactionHash}"),
                          ],
                        );
                      }
                    },
                  ),
                );
              }
            }
            ```
        </Tabs.Tab>
    </Tabs>
